# 3D脱出ゲーム「地下駐車場」

Unreal Engine5 で3D脱出ゲームを制作しました．

デモ動画とコードと投稿サイトのURLを記載しています．

ブループリントで実装したので，テキストファイルにエキスポートしたものをScriptsフォルダにまとめています．

# 注意点

解像度は1920×1080でプレイするようにしてください．

解像度1920×1200でプレイすると画面が見切れてしまいます．

また，マウスの使用を推奨します．
